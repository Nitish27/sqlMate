1. First, read `src/store/databaseStore.ts` fully to identify every state field and action. Categorize each into one of the 6 domains:

   **connectionStore:** savedConnections, openConnectionIds, activeConnectionId, connect(), disconnect(), addConnection(), updateConnection(), removeConnection(), setActiveConnectionId()

   **workspaceStore:** tabs, activeTabId (or activeTabIds per connection), openTab(), closeTab(), setActiveTab(), updateTab(). CRITICAL: Do NOT store row data (`rows?: any[][]`) in this store. Row data should remain in component-local refs (useRef) or a Map outside Zustand. If the current databaseStore stores rows in the tab object, strip that field during migration.

   **schemaStore:** sidebarItems (per connection), fetchSidebarItems(). In Phase 1 this is minimal — it grows significantly in Phase 3 when schema introspection is added.

   **historyStore:** queryHistory[], addToHistory(). Persisted via storeService.getHistory()/setHistory(). Ring buffer: cap at 500 items.

   **settingsStore:** theme, safeMode, fontSize, fontFamily, etc. In Phase 1 this is mostly placeholder defaults — it grows in Phase 2 when the preferences UI is built. Persisted via storeService.getSetting()/setSetting().

   **uiStore:** showConnectionModal, showDatabaseSelector, showImportDialog, showExportDialog, selectedDatabase, activePanels, notifications. Ephemeral — no persistence.

2. Create each store file using Zustand `create()`:

   **connectionStore.ts:**
   ```typescript
   import { create } from 'zustand';
   import { storeService } from '../services/StoreService';

   interface ConnectionState {
     savedConnections: SavedConnection[];
     openConnectionIds: string[];
     activeConnectionId: string | null;
     // ... all connection actions
     loadConnections: () => Promise<void>;  // called at app startup
     addConnection: (conn: SavedConnection) => Promise<void>;
     // ...
   }

   export const useConnectionStore = create<ConnectionState>((set, get) => ({
     savedConnections: [],
     openConnectionIds: [],
     activeConnectionId: null,
     loadConnections: async () => {
       const connections = await storeService.getConnections();
       set({ savedConnections: connections });
     },
     addConnection: async (conn) => {
       const next = [...get().savedConnections, conn];
       set({ savedConnections: next });
       await storeService.setConnections(next);
     },
     // ... extract remaining connection actions from databaseStore
   }));
   ```

   Follow this pattern for all 6 stores.

3. Create shared types file `src/store/types.ts` for interfaces shared across stores (SavedConnection, Tab, SidebarItem, HistoryItem, etc.). Stores import types from this file, NOT from each other. This prevents circular imports (Pitfall 3 from RESEARCH).

4. Create `src/hooks/useConnectionActions.ts` for cross-store coordination:
   ```typescript
   export function useConnectionActions() {
     const { connect: storeConnect, setActiveConnectionId } = useConnectionStore();
     const { fetchSidebarItems } = useSchemaStore();
     const { setShowConnectionModal } = useUIStore();
     const { openTab } = useWorkspaceStore();

     const connect = async (conn: SavedConnection, password?: string) => {
       await storeConnect(conn, password);
       setActiveConnectionId(conn.id);
       await fetchSidebarItems(conn.id);
       setShowConnectionModal(false);
       // Open a default query tab for the new connection
       openTab({ connectionId: conn.id, type: 'query', name: 'Query 1' });
     };

     return { connect };
   }
   ```

5. Put shared Tauri invoke wrappers in `src/services/` (or reuse existing patterns). The stores should call `invoke()` for backend operations but NOT directly — use thin wrapper functions.

6. Ensure NO store imports from another store directly. All cross-store reads happen in React components via multiple `useXxxStore()` calls or in coordinator hooks.

   Example migration:
   ```typescript

   function Sidebar() {
     const { sidebarItems, activeConnectionId, showConnectionModal } = useDatabaseStore();
     // ...

   import { useConnectionStore } from '../store/connectionStore';
   ```
